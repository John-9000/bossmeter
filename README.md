# Boss Meter Website

Official website for Boss Meter at [bossmeter.app](https://bossmeter.app).

This project contains the public website used to promote the Boss Meter app and host supporting pages such as privacy information and contact details.

## Main Files

- `index.html` - main Boss Meter website page
- `commercials.html` - commercials page with the website ad slot
- `sharedscore.html` - shared Boss Score page
- `privacy.html` - website privacy notice and app privacy policy
- `about.html` - about page
- `contact.html` - contact page
- `script.js` - main website logic
- `styles.css` - website styling
- `content.js` - quotes, funny texts, and shared content

## Ads

- `commercials.html` contains the website AdSense ad unit
- `ads.txt` is used for website ad authorization
- `app-ads.txt` is used for app ad authorization
- Google Privacy & messaging is used for website consent handling

## Notes

- The site includes Boss Inspiration, Boss Score rolling, score sharing, and the commercials page
- Website content is related to the Boss Meter Android app, but the website and app privacy sections are kept separate in `privacy.html`
